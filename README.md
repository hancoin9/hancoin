# HANCOIN | 汉币

**简介**  
HANCOIN（汉币）是一款极简、高速、免费、隐私、传承汉文化的数字资产。  
- 零手续费水龙头，每日免费领取汉币  
- 极速Rust后端，P2P账本同步，纯内存极致并发  
- 前端单文件、零依赖、移动优先，LOGO汉风极简（圆圈内大号“Ħ”）  
- 端到端隐私私聊、阅后自焚、朋友圈、超级群聊  
- 安全加固，所有敏感数据本地存储、签名认证、严格校验  
- 易用、极简、低资源，人人可一键部署和使用
